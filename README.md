WindowManagerSample
===================
skypeのような通話アプリの着信時の動作を検討するプロジェクトです。

## 着信時の要件
- アプリ利用時、アプリの前面に着信画面が出る
- ロック時、ロック画面の前面に着信画面が出る
- ロック画面で着信画面がでてるときに、ロック解除を(なんらかの方法で)したとき、着信画面が出る
- 画面OFF時、画面がONになり着信画面が出る

## 着信画面の要件
- ボタンが押せる

## やらないこと  
以下は着信画面表示をキャンセルするので検証しない
- 電話中の着信
- skype等の通話アプリで通話中の着信
